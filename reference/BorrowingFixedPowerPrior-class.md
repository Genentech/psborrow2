# `BorrowingFixedPowerPrior` class

`BorrowingFixedPowerPrior` class

## Slots

- `method_name`:

  string. The name of the method.

- `ext_flag_col`:

  character. Name of the external flag column in the matrix.

- `powers`:

  numeric. The fixed values to be used as the powers in the power prior.
