# Combine objects in `psborrow2`

Creates `data.frame` objects from various classes in `psborrow2`

## Usage

``` r
# S4 method for class 'SimDataList'
c(x, ...)
```

## Arguments

- x:

  object of type:
  [SimDataList](https://genentech.github.io/psborrow2/reference/SimDataList-class.md)

- ...:

  additional objects to combine

## Value

A combined object

list of
[`SimDataList`](https://genentech.github.io/psborrow2/reference/SimDataList-class.md)
objects.
