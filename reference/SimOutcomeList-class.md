# `SimOutcomeList` Class

A class for outcome details as part of a simulation study. Objects of
class `SimOutcomeList` should not be created directly but by the
constructor
[`sim_outcome_list()`](https://genentech.github.io/psborrow2/reference/sim_outcome_list.md).

## Slots

- `outcome_list`:

  named list of object of class `Outcome`, one object for each parameter
  variation.
