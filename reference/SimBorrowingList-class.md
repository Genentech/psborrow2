# `SimBorrowingList` Class

A class for borrowing details as part of a simulation study. Objects of
class `SimBorrowingList` should not be created directly but by the
constructor
[`sim_borrowing_list()`](https://genentech.github.io/psborrow2/reference/sim_borrowing_list.md).

## Slots

- `borrowing_list`:

  named list of object of class `Borrowing`, one object for each
  parameter variation.
