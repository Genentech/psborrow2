# Legacy function for the normal half prior

Please use
[`prior_half_normal()`](https://genentech.github.io/psborrow2/reference/prior_half_normal.md)
instead.

## Usage

``` r
half_normal_prior(...)
```

## Arguments

- ...:

  Deprecated arguments to `half_normal_prior()`.

## Value

This function does not return a value. When called, it triggers an error
message indicating that `half_normal_prior()` is deprecated and that
[`prior_half_normal()`](https://genentech.github.io/psborrow2/reference/prior_half_normal.md)
should be used instead.
