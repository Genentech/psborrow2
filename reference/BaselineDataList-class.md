# Baseline Data Frame List

A named `list` of `BaselineDataFrame`s with generated data for
`internal_treated`/`internal_control`/ `external_control` groups

## Value

A `BaselineDataList`

## Slots

- `baseline_object`:

  Simulated covariates definitions as `BaselineObject`. See
  [`create_baseline_object()`](https://genentech.github.io/psborrow2/reference/create_baseline_object.md)
