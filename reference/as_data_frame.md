# Coerce a `psborrow2` object to a data frame

Creates `data.frame` objects from various classes in `psborrow2`

## Usage

``` r
# S3 method for class 'BaselineDataList'
as.data.frame(x, ...)
```

## Arguments

- x:

  object of type:
  [BaselineDataList](https://genentech.github.io/psborrow2/reference/BaselineDataList-class.md)

- ...:

  Optional arguments for passed to
  [data.frame](https://rdrr.io/r/base/data.frame.html)

## Value

A `data.frame`
