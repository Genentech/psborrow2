# `SimTreatmentList` Class

A class for treatment details as part of a simulation study. Objects of
class `SimTreatmentList` should not be created directly but by the
constructor
[`sim_treatment_list()`](https://genentech.github.io/psborrow2/reference/sim_treatment_list.md).

## Slots

- `treatment_list`:

  named list of object of class `Treatment`, one object for each
  parameter variation.
