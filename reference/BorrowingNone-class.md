# `BorrowingNone` class

A class for defining details for "No borrowing" methods. Objects of
class `BorrowingNone` should not be created directly but by the
constructor
[`borrowing_none()`](https://genentech.github.io/psborrow2/reference/borrowing_none.md).

## Slots

- `method_name`:

  string. The name of the method.

- `ext_flag_col`:

  character. Name of the external flag column in the matrix.
