# Legacy function for binary logistic regression

Please use
[`outcome_bin_logistic()`](https://genentech.github.io/psborrow2/reference/outcome_bin_logistic.md)
instead.

## Usage

``` r
logistic_bin_outcome(...)
```

## Arguments

- ...:

  Deprecated arguments to `logistic_bin_outcome`.

## Value

This function does not return a value. When called, it triggers an error
message indicating that `logistic_bin_outcome()` is deprecated and that
[`outcome_bin_logistic()`](https://genentech.github.io/psborrow2/reference/outcome_bin_logistic.md)
should be used instead.
