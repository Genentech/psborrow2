# Legacy function for the exponential prior

Please use
[`prior_exponential()`](https://genentech.github.io/psborrow2/reference/prior_exponential.md)
instead.

## Usage

``` r
exponential_prior(...)
```

## Arguments

- ...:

  Deprecated arguments to `exponential_prior()`.

## Value

This function does not return a value. When called, it triggers an error
message indicating that `exponential_prior()` is deprecated and that
[`prior_exponential()`](https://genentech.github.io/psborrow2/reference/prior_exponential.md)
should be used instead.
