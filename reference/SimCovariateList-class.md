# `SimCovariateList` Class

A class for covariate details as part of a simulation study. Objects of
class `SimCovariateList` should not be created directly but by the
constructor
[`sim_covariate_list()`](https://genentech.github.io/psborrow2/reference/sim_covariate_list.md).

## Slots

- `covariate_list`:

  named list of object of class `Covariate`, one object for each
  parameter variation.
